# 🚀 QUICK SETUP GUIDE - ADmyBRAND Insights Dashboard

## ⚡ Instant Setup (3 Steps)

1. **Extract the ZIP file**
   ```
   Unzip: admybrand-insights-complete.zip
   ```

2. **Open in VS Code**
   ```
   cd admybrand-insights-fixed
   code .
   ```

3. **Run in terminal (Ctrl + `)**
   ```bash
   npm install
   npm run dev
   ```

4. **Open browser**: http://localhost:3000

## ✅ What You Get

- ✨ **Modern Analytics Dashboard** with interactive charts
- 🌓 **Dark/Light Theme Toggle** with persistent settings  
- 📊 **4 Chart Types**: Line, Area, Bar, and Pie charts
- 📱 **Responsive Design** works on all devices
- 🔍 **Advanced Data Table** with sorting, filtering, CSV export
- ⚡ **Real-time Updates** simulation (toggle on/off)
- 🎭 **Smooth Animations** using Framer Motion
- 💎 **Glassmorphism UI** with modern card effects

## 🛠️ Tech Stack

- **Next.js 14** + TypeScript + App Router
- **Tailwind CSS** for styling
- **Recharts** for data visualization  
- **Framer Motion** for animations
- **Lucide React** for icons
